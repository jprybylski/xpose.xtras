# An `xp_xtras` example of an M3 model (dataset)

The dataset used to fit the
[`pkpd_m3`](https://jprybylski.github.io/xpose.xtras/reference/pkpd_m3.md)
model.

## Usage

``` r
pkpd_m3_df
```

## Format

### `xp_xtras`

An `xp_xtras` object.
