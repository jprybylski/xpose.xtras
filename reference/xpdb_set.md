# An example `xpose_set` object

A set of identical `xpdb` objects to demo various features of
`xpose.xtras`.

## Usage

``` r
xpdb_set
```

## Format

### `xpose_set`

An `xpose_set` object of length 4 with a single lineage.

## Source

Assembled from the `xpdb_ex_pk` object in the `xpose` package.
