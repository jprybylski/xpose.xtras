# A more complex example of `xpose_set` object

Model-building set for the phenobarbital in neonates PK data used across
multiple packages.

## Usage

``` r
pheno_set
```

## Format

### `xpose_set`

An `xpose_set` object of length 14 with a branched lineage.

## Source

[doi:10.1159/000457062](https://doi.org/10.1159/000457062) and
nlmixr2data::pheno_sd

## Details

This is not a demonstration of high-quality model-building, it is just a
typical and simple example.
