# An `xp_xtras` example of a final model

Final model for phenobarbital in neonates.

## Usage

``` r
pheno_final
```

## Format

### `xp_xtras`

An `xp_xtras` object.

## Source

[doi:10.1159/000457062](https://doi.org/10.1159/000457062) and
nlmixr2data::pheno_sd

## Details

This is re-parameterized from the covariate-building work, which in this
case did not identify a relationship with Apgar score.

This is `run16` in
\<[`pheno_set`](https://jprybylski.github.io/xpose.xtras/reference/pheno_set.md)\>
