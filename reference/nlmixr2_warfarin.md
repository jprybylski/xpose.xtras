# An `xp_xtra` example based on a complex nlmixr2 fit

Based on the multiple endpoint warfarin PK/PD example.

## Usage

``` r
nlmixr2_warfarin
```

## Format

An object of class `xp_xtras` (inherits from `xpose_data`, `uneval`) of
length 10.

<https://nlmixr2.org/articles/multiple-endpoints.html>
