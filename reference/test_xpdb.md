# Logical instead of exception for xpose data check

Logical instead of exception for xpose data check

## Usage

``` r
test_xpdb(xpdb, check = "data")
```

## Arguments

- xpdb:

  An xpose database object.

- check:

  The \`xpdb\` slot to be checked, can be \`data\`, \`files\`,
  \`summary\`, \`special\`, \`code\`, a file name, or \`FALSE\`. IF
  \`FALSE\` the slot test will be skipped.
