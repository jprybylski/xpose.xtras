# An `xp_xtra` example based on an nlmixr2 fit

This is the most basic one compartment example used in nlmixr2 example
documentation. It is a fit to the popular theophylline dataset.

## Usage

``` r
xpdb_nlmixr2

xpdb_nlmixr2_saem
```

## Format

An object of class `xp_xtras` (inherits from `xpose_data`, `uneval`) of
length 10.

An object of class `xp_xtras` (inherits from `xpose_data`, `uneval`) of
length 10.
